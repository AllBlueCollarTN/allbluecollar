FREE PUBLISHING OPTIONS

GitHub Pages:
1. Create a free GitHub account.
2. Create a new public repository named all-blue-collar-website.
3. Upload every file and the assets folder.
4. Open Settings > Pages.
5. Under Build and deployment, choose Deploy from a branch.
6. Select main and /root, then Save.
7. Add your purchased domain under Custom domain.
8. Follow the DNS instructions shown by GitHub at your domain registrar.

Cloudflare Pages:
1. Create a free Cloudflare account.
2. Open Workers & Pages > Create > Pages > Upload assets.
3. Upload this folder or ZIP file.
4. Open Custom domains and add your purchased domain.
5. Follow Cloudflare's DNS prompts.
