ALL BLUE COLLAR WEBSITE UPDATE

To replace the current GitHub Pages site:
1. Open the allbluecollar repository on GitHub.
2. Click Add file > Upload files.
3. Upload index.html, styles.css, script.js and the assets folder from this package.
4. Choose "Commit directly to the main branch" and click Commit changes.
5. Wait 1-3 minutes and refresh allbluecollar.com.

Do not upload the ZIP itself. Upload the files inside the unzipped folder.

The estimate form opens an email addressed to info@allbluecollar.com.
Project gallery boxes are placeholders until real job photos are available.
